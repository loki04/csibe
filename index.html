Code Size Benchmark

CSiBE is a code size benchmark environment which can help to compact your project or the generated code of your compiler.

Use it Now

Code size metrics

The primary purpose of CSiBE is to monitor the size of the code generated by compilers. In addition, compilation time and code performance measurements can also be done.

On both side of the road

You can use CSiBE to monitor your compiler and the progress of code compaction optimization.

On the other hand it is possible to observe the code size changes in your project with the help of CSiBE.

Try it!

Free and Open Source

CSiBE is free and open source. Every change is visible at the project's page. If you wish you can also contribute to it.
